## 📖 Overview

<a href="#iExplorer">iExplorer</a> serves Mac users who need more file access to their iPhone than Apple's standard tools provide — users who want to copy music off their phone, retrieve specific files from iOS apps, access messages from the desktop, or manage device files without the sync-based restrictions of iTunes and Finder.

The <a href="#iExplorer">music recovery use case</a> is iExplorer's most common individual use — recovering music from an iPhone when the computer that originally synced the library is no longer available, transferring the music library to a new Mac, or copying purchased or recorded music off the device that iTunes won't transfer in the normal sync direction. <a href="#iExplorer">File retrieval from iOS apps</a> uses iExplorer when a specific app doesn't provide its own export mechanism — extracting a document from an app sandbox, retrieving a recording from a specialized audio app, or accessing data that a deleted app left behind.

The <a href="#iExplorer">messages export use case</a> recovers or archives iPhone text message and iMessage history — exporting conversations for legal purposes, archiving important messages, or recovering message content from a device before replacement or disposal. <a href="#iExplorer">Developer and IT use</a> applies iExplorer for accessing app sandbox contents during iOS app development and testing, retrieving diagnostic files from managed iOS devices, and the file system access that device management tasks sometimes require.</p>

<a href="#iExplorer">Backup data extraction</a> retrieves specific data from iPhone backups without full restore — extracting a specific photo from a two-year-old backup, recovering messages from before a device reset, or getting specific app data from a backup without overwriting the current device state. <a href="#iExplorer">No iTunes sync requirement</a> handles file transfer without triggering an iTunes sync — accessing individual files and data without the risk of sync overwriting device content.

<a href="#iExplorer">Multiple device support</a> manages and switches between multiple connected iOS devices — useful for users with multiple Apple devices or for IT scenarios with several devices to manage. <a href="#iExplorer">Macroplant development</a> from an established Mac software developer with iExplorer as a long-running product provides the macOS compatibility updates and continued development that iOS device management tools require as iOS evolves.</p>

The backup browsing capability addresses a use case that Apple's own restore mechanism doesn't support: selective data extraction from a backup. Apple's backup restore replaces the entire device content with backup content — a complete overwrite that isn't appropriate when the user only needs one file, one conversation, or one photo from an old backup. iExplorer reads the backup data and exposes it as browseable content, allowing the specific extraction without disrupting the current device state.
